# Installation

Clone the repo:

git clone https://github.com/yourusername/mindEase-App.git
cd mindEase-App


Install dependencies:

npm install


Run the app:

npx expo start


Build for Android:

eas build --platform android


Install on your device by scanning the QR code or using the generated APK/AAB.
